##  Overview
A modern, responsive bakery website showcasing a welcoming interface with sections:
- **Home**, **About**, **Our Offer**, **Gallery**, **Blog**, **Contact**  
- Fully responsive navbar and layout  
- Beautiful imagery to entice visitors
